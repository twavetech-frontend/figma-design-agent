### C. 송금 플로우 (Send Money — 5단계)

**플로우**:
```
수신자 선택 → 계좌번호 입력 → 은행 선택 → 금액 입력 → 확인
```

**레이아웃 패턴** (각 단계 공통):
```
┌─ Status Bar ────────────────────────┐
├─ Back (<) ──────────────────────────┤
├─ 질문 제목: "어디로 돈을 보낼까요?" ┤  ← 큰 Bold
├─ [세그먼트 or 입력 필드] ───────────┤
│                                     │
│         (콘텐츠 영역)               │
│                                     │
├─ 커스텀 넘버패드 (최하단) ──────────┤
│  1   2   3                          │
│  4   5   6                          │
│  7   8   9                          │
│  00  0   ←                          │
└─────────────────────────────────────┘
```

**단계별 특징**:
1. **수신자 선택**: 탭(계좌/연락처) + 리스트
2. **계좌 입력**: 텍스트 필드 + 넘버패드
3. **은행 선택**: 바텀시트, 3열 그리드 (16개 은행)
4. **금액 입력**: 중앙 큰 숫자 + 넘버패드
5. **확인**: 요약 카드 + Blue [다음] 버튼

**핵심 패턴**:
- 질문형 UX: "~를 ~할까요?" 패턴 일관 적용
- 넘버패드: `00` / `0` / `←(delete)` 3개 하단, 폰트 크기 20px
- 바텀시트: `cornerRadius: 16px` top-only, 상단 drag handle (4×32px bar)

## 6. 인터랙션 & UX 패턴

### 질문형 플로우 (Conversational Flow)
Julee의 핵심 UX 패턴 — 사용자에게 한 번에 하나만 질문

```
"어디로 보낼까요?" → 선택
      ↓
"얼마나 보낼까요?" → 입력
      ↓
"이렇게 보낼까요?" → 확인 카드
      ↓
"보냈어요!" → 완료 체크 + 단일 CTA
```

### 프로그레시브 디스클로저
- 단계마다 하나의 입력 필드만 노출
- 이전 단계 정보는 헤더나 요약 카드로 표시
- 다음 단계 버튼은 입력 완료 시 활성화

### 성공/완료 화면
- 큰 체크(✓) 아이콘 — Blue 원형 배경, 72px
- 굵은 완료 메시지 (24px Bold)
- 서브 텍스트 (부가 정보, 14px Gray)
- 단일 [확인] CTA (Full-width Blue)

### 다크/라이트 모드
- 시스템 설정 연동 지원
- 홈, 증권 등 주요 화면 다크 테마 최적화
- 카드 bg 대비: Light #F9FAFB vs Dark #1B1D1F

## 8. 디자인 생성 시 적용 포인트

Julee 스타일로 DS v1 기반 화면 설계할 때:

1. **질문형 헤더** — 제목을 "~을 하시겠어요?" 형식으로 작성
2. **금액 표시** — `Bold 28px` + DS `Typography/Numeric/XL` 스타일
3. **CTA 배치** — 화면 최하단 fixed, Full-width, `height: 52px`
4. **카드 컨테이너** — `cornerRadius: 16px`, DS `Elevation/1` 이펙트
5. **거래 리스트** — 아이콘 36px + 텍스트 left + 금액 right align
6. **완료 화면** — 중앙 정렬, 큰 체크 아이콘, 단일 CTA
7. **다크 테마** — bg #17171C, 카드 #1B1D1F (DS 다크 토큰 사용)
